# G2-Concurrency-Race-Condition-Lab-Python
Laboratorio práctico en Python 3.13 para demostrar cómo la concurrencia puede mejorar el rendimiento de un sistema, pero también provocar condiciones de carrera, datos inconsistentes y errores difíciles de detectar cuando varios hilos modifican estado compartido sin control. Incluye un ejemplo incorrecto y una solución segura usando threading.Lock.
